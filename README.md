# 🛍️ E-commerce Application

Welcome to the **E-commerce App** repository!
This is a modern, TypeScript-powered web application designed for a smooth and scalable online shopping experience. Whether you're browsing or building on it, you're in good hands.

---

## 🚀 Getting Started

Ready to spin this up on your machine? Follow these simple steps to get rolling.

### ✅ Prerequisites

Make sure you've got the following installed:

* [Node.js](https://nodejs.org/) (v14 or higher recommended)
* [npm](https://www.npmjs.com/) (Node Package Manager)

### 📦 Installation

1. Clone the repo:

   ```bash
   git clone https://github.com/ArchitSaxena349/ecommerce.git
   ```

2. Move into the project directory:

   ```bash
   cd ecommerce
   ```

3. Install all dependencies:

   ```bash
   npm install
   ```

---

## 🛠️ Usage

Once the setup is done, fire up the development server:

```bash
npm start
```

This will launch the app in your default web browser, typically at:

```
http://localhost:3000
```

Make changes → Save → See them live. Easy.
